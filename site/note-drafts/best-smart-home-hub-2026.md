# スマートホームハブおすすめ2026：ネット障害でも動く家を作る選び方

スマートホームハブを選ぶ時、最初に確認すべきは「インターネットが落ちたとき何が動かなくなるか」だ。Google Nest Hub Maxのオートメーションは全てクラウド依存——Google障害でスケジュール照明も全停止する。Aeotec Smart Home HubはZ-WaveとZigbeeをローカル処理するため、ネット回線が死んでも自動化は継続する。Amazon Echo Hubは8インチ壁面パネルとZigbee/Thread/Matter対応で最もこなれたUIを持つが、Z-Wave非対応が既存スマートロックユーザーに制約となる。

## 結論

**総合ベスト**: Amazon Echo Hub（新規構築・Matter中心）またはAeotec Smart Home Hub（Z-Wave既存機器混在・ローカル処理重視）。

## 選んだ理由

- Aeotecはローカル処理でネット障害時も自動化継続——最も信頼性が高い
- Echo Hubは8インチ壁面パネルで視認性・操作性が他の追随を許さない
- Apple HomePod miniはHomeKitをローカル完結、Thread Border Router兼任で¥12,800

## こんな人向け

**Matter/Thread対応デバイスだけの新規構築**: Echo HubがUI・対応プロトコル・デザインで最良
**古いZ-Waveデバイス（スマートロック等）が混在**: AeotecかSmartThings StationでZ-Wave+Zigbee+Matter全対応
**iPhone/Mac中心のAppleエコシステム**: HomePod mini一択でHomeKitローカル完結
