## サイト紹介文(短)100字以内

```
17言語で運営する商品比較・レビューサイト。VPN、ガジェット、家電、生活用品を実際にテスト・比較し、Pinterestを主な流入元として読者に役立つ情報を発信。
```

## サイト紹介文(中)200字以内

```
Picklyは商品の実テストと正直な比較を17言語で発信する独立系メディアです。ガジェット・家電・生活用品・金融サービス等を「価格」「速度」「使い勝手」など定量的指標で評価し、Pinterestや検索を経由して訪れた読者に購入判断の材料を提供します。各記事には必ず弱点・トレードオフを明記し、編集と広告を分離しています。
```

## サイト紹介文(長)400-500字

```
Picklyは独立系の編集型レビュー・比較メディアです。日本拠点・個人事業として運営し、英語版を出発点に日本語、中国語(簡体・繁体)、韓国語、スペイン語、ポルトガル語(ブラジル)、フランス語、ドイツ語、イタリア語、ロシア語、アラビア語、ヒンディー語、インドネシア語、タイ語、ベトナム語、トルコ語の合計17言語で配信しています。

主要なジャンルはガジェット・家電、ホーム/インテリア、フィットネス、ファイナンス、トラベルなど日常的な購買判断が発生する領域で、各記事は価格・速度・耐久性等の定量指標を用いた検証を行い、必ず1つ以上の弱点や代替案を明記しています。AdSense等のディスプレイ広告ではなくアフィリエイトを主な収益源とし、Pinterest経由の流入を主軸に置いています。

掲載商品の選定は編集独自の基準で行い、報酬の高低が順位に影響することはありません。すべてのアフィリエイトリンクには「PR」表示と rel="sponsored" を付与し、景品表示法・特定商取引法・各ASP規約への準拠を担保しています。
```

## 主要ジャンル(ASP の登録カテゴリで近いものを選ぶ)

優先順:

1. **比較・ランキング** (最頻出)
2. **商品レビュー**
3. **ガジェット / IT / インターネット**
4. **生活情報 / ライフハック**
5. **ファイナンス / 金融** (該当記事ある時のみ)
6. **美容 / ヘルスケア** (該当記事ある時のみ)
7. **旅行**

## 個人情報(チャットには貼らない、申請時に直接入力)

ASP申請で必要となる個人情報。手元の本人確認資料と一致させる:

- 氏名(漢字)
- 氏名(フリガナ)
- 生年月日
- 郵便番号
- 都道府県
- 市区町村
- 番地
- 建物名・部屋番号
- 電話番号
- 連絡先メールアドレス(`contact@pickly.blog` 推奨、または個人 Gmail)
- 銀行口座(振込用、本人名義必須)
  - 銀行名
  - 支店名
  - 口座種別(普通)
  - 口座番号
  - 名義(カタカナ)
